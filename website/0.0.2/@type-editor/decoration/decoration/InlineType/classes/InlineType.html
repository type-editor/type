Class: InlineType ​

Defined in: decoration/InlineType.ts:19

Inline decoration type that applies styling or attributes to a range of content without changing the document structure.

Inline decorations are used to add visual styling or data attributes to a range of content, such as highlighting search results, showing tracked changes, or marking spelling errors. They render as inline elements (like <span>) that wrap the decorated content.
